## 📱 تصميم متجاوب (Responsive)

### للشاشات الصغيرة (< 360px)

```
- تقليل padding إلى 15px
- تصغير زر + إلى 120px
- تقليل حجم الخط الرئيسي إلى 64px
- إخفاء الآية في الرأس
```

### للشاشات المتوسطة (360-480px)

```
- padding: 20px (افتراضي)
- زر +: 150px
- الأحجام الافتراضية
```

### للشاشات الكبيرة (> 480px)

```
- padding: 25px
- زر +: 180px
- زيادة المسافات بين العناصر
- عرض الإحصائيات في صفين
```

## 🌙 التصميم الإسلامي

### العناصر الإسلامية في التصميم

```
1. الألوان:
   ✓ أزرق داكن (السماء والروحانية)
   ✓ أخضر (الإسلام والنماء)
   ✓ ذهبي/برتقالي (الدفء والإيمان)

2. الخطوط:
   ✓ واضحة وكبيرة (سهولة القراءة)
   ✓ RTL كامل (العربية من اليمين)
   ✓ تباعد مناسب (راحة البصر)

3. الواجهة:
   ✓ بسيطة وغير مزدحمة
   ✓ تركيز على الوظيفة
   ✓ بدون زخارف مشتتة
```

## 🎨 Color Psychology (سيكولوجية الألوان)

### الألوان ومعانيها في التطبيق

```
🔵 الأزرق الداكن (#0d1b2a)
   → الهدوء والسكينة
   → الروحانية والتأمل
   → الاحترافية

🟢 الأخضر (#4CAF50)
   → الإيجابية والنمو
   → الإسلام والطبيعة
   → النجاح والإنجاز

🟠 البرتقالي (#FF9800)
   → الحذر الخفيف
   → الطاقة والحيوية
   → الانتباه

🔴 الأحمر (#f44336)
   → التحذير القوي
   → الأهمية العالية
   → طلب التأكيد
```
